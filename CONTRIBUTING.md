# Contributing

Grazie per l'interesse nel contribuire a **Meeting Notes Agent**.

## Come contribuire
- Apri una issue descrivendo il bug o la feature proposta.
- Se vuoi lavorarci, crea una branch e apri una pull request.
- Mantieni le modifiche piccole e facili da revisionare.

## Setup locale

Backend:
```bash
cd backend
mvn spring-boot:run
```

Frontend:
```bash
cd frontend
npm install
npm run start
```

## Linee guida
- Rispetta la privacy: non inserire dati sensibili nei test o nelle demo.
- Evita dipendenze non necessarie.
- Aggiungi note chiare se modifichi la pipeline.

## Licenza
